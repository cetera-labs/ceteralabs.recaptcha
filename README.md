# Модуль Bitrix - ceteralabs.recaptcha

## Описание
Модуль автоматически заменяет обычную капчу 1С-Битрикс на [Google reCAPTCHA](https://developers.google.com/recaptcha/intro?hl=ru).
* Поддержка всех форм сайта: регистрация, авторизация, обратная связь;
* Поддержка нескольких форм на одной странице;
* Поддержка ajax-форм;

__Если на странице отсутствуют формы с капчей, дополнительные скрипты не будут подключаться!__

### Получение ключей:
Авторизоваться через google аккаунт и зарегистрировать свой сайт по [ссылке](https://www.google.com/recaptcha/admin/create) и получить ключи.

### Настройка модуля:
В административном разделе  ( Настройки > Настройки продукта > Настройки модулей > CeteraLabs reCAPTCHA ) вставить полученные ключи.

### Дополнительно:
В настройках можно изменить подпись и стандартное сообщение об ошибке.
